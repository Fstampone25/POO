# 👋 Repositorio de Facundo Stampone

## Programación Orientada a Objetos

Bienvenido/a a mi repositorio de la materia **Programación Orientada a Objetos**. Acá voy a ir subiendo los trabajos prácticos y proyectos que desarrolle a lo largo de la cursada, utilizando **Java** como lenguaje principal.

Más allá de cumplir con las entregas, mi objetivo con este repo es también llevar un registro de mi progreso como programador: cada trabajo es una oportunidad para afianzar los conceptos fundamentales de la POO (encapsulamiento, herencia, polimorfismo, abstracción) y seguir mejorando la forma en la que escribo y organizo código.

## 🎯 Sobre este espacio

- ☕ Proyectos y ejercicios desarrollados en **Java**
- 🧠 Aplicación práctica de los pilares de la Programación Orientada a Objetos
- 🏗️ Diseño de clases, relaciones entre objetos, e implementación de estructuras cada vez más complejas
- 🚀 Un lugar para experimentar, equivocarme y mejorar con cada entrega
